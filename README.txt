Employee Attendance Management System (VBA) - Setup Instructions
===============================================================

This folder contains:
- Employee_Attendance_Template.xlsx -> The attendance register template.
- VBA_Code.txt -> VBA code to be pasted into the VBA editor.

Steps to complete your project in Excel 2007:

1. Open Employee_Attendance_Template.xlsx in Microsoft Excel 2007.
2. Press ALT + F11 to open the VBA Editor.
3. Insert a new Module: Right-click on any item in the Project Explorer → Insert → Module.
4. Paste the code from VBA_Code.txt into the module.
5. Close the VBA editor.
6. Enable the Developer tab: (Office Button > Excel Options > Popular > Show Developer tab)
7. Insert two Form Control buttons:
   - One for "Mark In" → Assign macro: MarkInTime
   - One for "Mark Out" → Assign macro: MarkOutTime
8. Save the file as: Excel Macro-Enabled Workbook (*.xlsm)

You can now start using the attendance system with In-Time and Out-Time tracking.
